# CareerLens

CareerLens is a Tech Career Intelligence Platform that analyzes Indian tech job markets.

## Features

- Scrapes tech job postings
- Stores job data in PostgreSQL
- Analyzes skill demand
- Shows hiring trends

## Tech Stack

- Python
- Flask
- PostgreSQL
- BeautifulSoup
- Selenium

├── app
